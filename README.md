# Republic Private Chat

## Overview
This repository contains a complete private chat application with backend and frontend setup using Docker.

## Getting Started

1. Clone the repository
2. Run `docker-compose up` to start the application
3. Access the application on `http://localhost`

## Technologies Used
- Node.js
- Express
- MongoDB
- React
- Docker

## File Structure
- backend/: Contains the backend server files
- frontend/: Contains the React files
- docker-compose.yml: Docker setup
- .gitignore: Files to ignore in git
